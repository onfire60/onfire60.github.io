Environmental Charter High School Site

Link to the site: ECHS Website

This project was completed for ECHS, a client of Overit Media. The purpose of this project was to create a brand new website for a school that teaches about the importance of the environment and conservation. The design had been taken care of by an outsourced designer who was unable to complete the development side of the project so he contracted Overit to do the development side of the job.

This project was completed entirely in Coldfusion except for the alumni blog portion of the site which used Wordpress backend with PHP. Each of the main site pages though were developed into separate page templates that were then linked into the custom Coldfusion backend system developed by Overit Media. This site, having a backend system, was developed to be dynamic so the client could update it easily. Each page template supported the overview of each section as well as the dynamic sub-pages of each section. Per the client’s request each page template then was coded to enable for video, photos, audio and downloads giving them the ultimate comparability for each page. Another thing that was checked and was very important was the cross browser compatibility of the site. The client was very happy with my skilled ability to perform that tasks he requested as well as doing them in a timely manner.

With all the work I put into this site I am extremely happy with the result. This site, aside from my current site, had the most work I've ever done for a client.
