Goodstuff Design Site

Link to site: Goodstuff Design Site

This was a project for a client of Overit Media. The purpose of this project was to develop a site for a designer for whom we occasionally contracted work. The site was to be developed in Coldfusion with Overit Media's custom backend system.

I started by cutting up the design; then I proceeded to create the page templates for the home and secondary pages. After these were completed, I created the templates for every other page that exist in the site. This site having a backend system was developed to be dynamic so the client could update it easily. The site was a simple construction, but it had to be cross browser compatible as well. After this site checked out under all major browsers, the site was then handed off to the client who was pleased that his site design turned out so well.
