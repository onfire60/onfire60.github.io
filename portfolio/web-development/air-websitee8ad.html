Air Website

Link to Site: Artists Image Resource Website

During college I got the opportunity to intern at a 14 year old non- profit screen printing organization called the Artists Image Resource. While I was there they needed someone to develop their website for them so I stepped up to the plate. Other than my own site, this was actually one of the first sites I ever developed for a client. The unfortunate thing about developing for this client was the fact that their server was pure HTML based only so no PHP was allowed to make page templates. Therefore, I had to code out each page individually, but you have to adapt to what you have to work with.

The main page was developed to be dynamic in that the home page background would be dynamic with every page reload. The issue with this was that the design had images on top of the backgrounds, so with the issues of IE6 browser alpha with PNG files I had to incorporate new techniques for it to be cross browser compatible. Although this was an issue at first IE script filters did the trick and it still works to this day because IE6 was and is such a pain to develop.

Although I left the site in good standing when I finished my internship, it has now been literally beaten into submission and looks awful, but at least I have screen shots of it in its original glory. Visit the link above at your own risk to your eyes. I sometimes think about calling them up to redo the site for them with a backend system so they can easily update the site, but I'd have to host them on my server because no PHP is just ridiculous nowadays.
