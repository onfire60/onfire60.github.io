Five Graces Alt Site

Link to the site (disable javascript to view): Five Graces Alt Site

This project was completed for a client of Overit Media. The purpose of the project was to create an alternate site for Five Graces hotels and suites. The reason they needed an alt site was the fact that the site was a full Flash site and by the off chance a user didn't have Flash installed this site wouldn’t be viewable.

This site was developed in Coldfusion and hooked into the Coldfusion backend system created by Overit Media. This site having a backend system was developed to be dynamic so the client could update it easily. I started by cutting up the design; then I proceeded to create the page templates for the home and secondary pages. After these were completed, I created the templates for every other page that exist in the site. The development was simple as it needed to be because without Flash the site couldn't be animated. Although not many people will see this site it’s still an intricate part to making the site user friendly and more compatible.
